# AdPages Copy Checker for Figma

A minimal Figma plugin surface for checking ad copy inside design files before it becomes campaign collateral.

The plugin runs locally in Figma. It does not make network requests, load remote assets, or require paid dependencies.

## What it does

- Reads selected text layers in Figma design files and sends their text into the plugin UI.
- Checks each non-empty line against Google Ads search copy length limits:
  - Headline target: 30 characters.
  - Description target: 90 characters.
- Accepts manually entered copy for the same checks.
- Generates UTM-tagged destination URLs from local form fields.

## Local Development

Compile the plugin controller:

```sh
npx tsc -p integrations/figma/adpages-copy-checker/tsconfig.json
```

Then import `integrations/figma/adpages-copy-checker/manifest.json` in Figma Desktop:

1. Open Figma.
2. Go to Plugins > Development > Import plugin from manifest.
3. Select this plugin's `manifest.json`.
4. Run `AdPages Copy Checker` from the development plugins menu.

## Publishing Notes

- The manifest intentionally uses `networkAccess.allowedDomains: ["none"]`.
- Replace the numeric placeholder `id` with the ID generated by Figma before publishing or updating through Community.
- Add final icons, screenshots, support URL, privacy copy, and a real Community description before public release.
- This checker only validates common copy length constraints. It does not guarantee Google Ads approval, policy compliance, or live account compatibility.
